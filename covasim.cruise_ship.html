covasim.cruise_ship package¶

Submodules¶

covasim.cruise_ship.model module¶

This file contains all the code for a single run of the cruise ship flavor of Covasim.

Based heavily on LEMOD-FP (https://github.com/amath-idm/lemod_fp).

Version: 2020mar20

Define age-sex distributions. Passenger age distribution based on:

https://www.nytimes.com/reuters/2020/02/12/world/asia/12reuters-china-health-japan.html

“About 80% of the passengers were aged 60 or over [=2130], with 215 in their 80s and 11 in the 90s, the English-language Japan Times newspaper reported.”
